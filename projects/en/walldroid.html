Walldroid Wallpaper

Android App

Walldroid is a great wallpaper app. Which has thousands of HD wallpapers of different categories. App offers number of features with great UI and design.

Offers free High-Definition beautiful images.
Easy to view and easy to set as wallpaper.

- High quality of wallpapers
- Categories with sort option
- Find by category
- Easy to set as wallpaper
- You can save image
- Add favorites
